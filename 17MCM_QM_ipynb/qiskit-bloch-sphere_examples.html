Bloch Sphere Examples

Bloch Sphere Examples#

To run these examples you need to install qiskit. The function hopf_map converts the state vector of a 1-qubit state to a vector in the 3D-Bloch Sphere. Remember that the Bloch Sphere is only a visualization, the vector in the sphere is not the state vector.

import math
from qiskit.visualization import plot_bloch_vector
%matplotlib inline

def hopf_map(state):
    x_0 = state[0].real
    x_1 = state[0].imag
    x_2 = state[1].real
    x_3 = state[1].imag
    x = 2 * (x_0 * x_2 + x_1 * x_3)
    y = 2 * (x_3 * x_0 + x_1 * x_2)
    z = x_0 ** 2 + x_1 ** 2 - x_2 ** 2 - x_3 ** 2
    return (x, y, z)

# plot |0>
plot_bloch_vector(hopf_map((1, 0)))

# plot |1>
plot_bloch_vector(hopf_map((0, 1)))

# plot |+>
plot_bloch_vector(hopf_map((1 / math.sqrt(2), 1 / math.sqrt(2))))

# plot |->
plot_bloch_vector(hopf_map((1 / math.sqrt(2), -1 / math.sqrt(2))))

# plot |+i>
plot_bloch_vector(hopf_map((1 / math.sqrt(2), 1.j / math.sqrt(2))))

# plot |-i>
plot_bloch_vector(hopf_map((1 / math.sqrt(2), -1.j / math.sqrt(2))))
